## Telegram English Bot

Дружелюбный Telegram-бот для практики английского языка на Python + OpenAI.

Пользователь пишет сообщения на английском, бот:
- **отвечает на английском**;
- **поддерживает свободный разговор**;
- **задаёт встречный вопрос**;
- **исправляет ошибки** и **кратко объясняет их по-русски**.

Формат ответа бота:

Ответ:  
\<естественный ответ на английском, с встречным вопросом\>

Исправления:  
❌ ошибка пользователя  
✅ правильный вариант  

Пояснение:  
\<короткое объяснение на русском\>

---

### 1. Требования

- Python **3.11+**
- Аккаунт в [OpenAI](https://platform.openai.com/)
- Созданный **OpenAI API ключ**
- Созданный **Telegram Bot Token** через `@BotFather`

---

### 2. Как получить OpenAI API ключ

1. Зайдите на сайт [OpenAI Platform](https://platform.openai.com/).
2. Войдите в аккаунт или зарегистрируйтесь.
3. Перейдите в раздел **API keys**.
4. Нажмите **Create new secret key**.
5. Скопируйте ключ и **сохраните его в надёжном месте** — после этого он больше не будет показан.

Этот ключ нужно будет записать в файл `.env`.

---

### 3. Структура проекта

`telegram_english_bot/`  
- `bot.py` — основной файл Telegram‑бота (точка входа)  
- `ai_service.py` — работа с OpenAI API (модель `gpt-4o-mini`)  
- `config.py` — загрузка переменных окружения из `.env`  
- `prompts.py` — системный промпт для AI (поведение преподавателя)  
- `requirements.txt` — список зависимостей  
- `.env.example` — пример файла переменных окружения  

Корень проекта:  
- `README.md` — эта инструкция

---

### 4. Настройка окружения

Перейдите в каталог проекта:

```bash
cd "English Bot"
```

Создайте виртуальное окружение (пример для Windows, PowerShell):

```bash
python -m venv .venv
.venv\Scripts\activate
```

Установите зависимости:

```bash
pip install -r telegram_english_bot/requirements.txt
```

---

### 5. Настройка `.env`

1. В каталоге `telegram_english_bot` создайте файл `.env` на основе `.env.example`:

```bash
cd telegram_english_bot
copy .env.example .env
```

2. Откройте `.env` и подставьте свои значения:

```env
OPENAI_API_KEY=sk-ваш-ключ-из-openai
TELEGRAM_BOT_TOKEN=ваш-бот-токен-из-botfather
```

---

### 6. Запуск бота

Из корня проекта (где лежит `README.md`) выполните:

```bash
cd "English Bot"
.venv\Scripts\activate
python -m telegram_english_bot.bot
```

Или:

```bash
python telegram_english_bot/bot.py
```

После запуска:
- найдите вашего бота в Telegram по имени;
- напишите ему любое сообщение **на английском**;
- бот ответит на английском, задаст встречный вопрос и внизу покажет блок с исправлениями и объяснением по‑русски.

---

### 7. Как это работает

- Бот использует библиотеку **aiogram** для работы с Telegram API.
- Библиотека **python-dotenv** подгружает переменные из `.env`.
- В `ai_service.py` бот отправляет историю диалога (последние 5 сообщений пользователя и ответы бота) в модель **`gpt-4o-mini`** через OpenAI API.
- В `prompts.py` хранится подробный системный промпт, который заставляет модель:
  - отвечать **только на английском**;
  - вести себя как **дружелюбный преподаватель английского**;
  - **мягко исправлять ошибки** и **кратко объяснять грамматику по‑русски**;
  - всегда соблюдать требуемый формат ответа.

---

### 8. Полезные подсказки

- Если бот не запускается — проверьте, что:
  - активировано виртуальное окружение;
  - корректно установлены зависимости;
  - в `.env` верно записаны `OPENAI_API_KEY` и `TELEGRAM_BOT_TOKEN`.
- Если бот молчит или выдаёт ошибку — подождите пару минут и попробуйте ещё раз: иногда OpenAI API временно недоступен.
